# System Design – Bharat Assist

## Architecture Overview
User → Mobile App / WhatsApp → AI Layer → Government Scheme Database → Response

## Components

1. User Interface
- Mobile-friendly
- Voice input/output

2. AI Layer
- NLP (Natural Language Processing)
- Multilingual LLM
- Eligibility matching logic

3. Database
- Government scheme data
- User interaction logs

4. Cloud Infrastructure
- AWS hosting

## Future Scope
- Offline mode
- Direct scheme application support
- Expansion to more languages
